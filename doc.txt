#19. Các keywords sử dụng với Redux
- actions: là 1 biến object
- dispatch: công cụ thực hiện hành động
- reducer: công cu update store
- store (state)
slide (reducer + action) theo logic thì slide sẽ nằm trong store

//slide quăng vô store



Tình huống	Diễn giải dễ hiểu
useEffect(() => {...})	Chạy sau mỗi lần render
useEffect(() => {...}, [])	Chạy 1 lần duy nhất khi component mount
useEffect(() => {...}, [value])	Chạy khi value thay đổi
return () => {...}	Chạy khi component bị huỷ (unmount)

Cú pháp	Nghĩa cụ thể

useState([])	users khởi tạo là một mảng rỗng []
